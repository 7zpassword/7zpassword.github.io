Microsoft edge 90 official release: new download management menu, clearer font rendering

 

The official version of Microsoft edge V90 was released on April 15. The new version of the browser added child mode and other functions, in addition to a number of performance improvements.
Microsoft edge 90 official release: new download management menu, clearer font rendering
The child model of edge 90 is suitable for children aged 5-8 and 9-12. After this mode is turned on, the security search function of Bing search will be automatically turned on to prevent children from contacting violence, pornography and other information. In addition, children's mode has a custom browser theme.

In terms of security, the new version of edge browser can avoid mining script attacks on Web pages, and at the same time, it can avoid cookies being used by other websites. For enterprise users, the browser can submit the risks found by app guard to the manager, and the operation and maintenance personnel can distribute specific security policies within the enterprise.
Here are the new features of edge 90 browser:
Download button
An independent download button is added to the top toolbar of the new browser, which is convenient for users to directly manage and search the downloaded files. At the same time, there is an option of "open file" under each file.

Pdf printing
Edge 90 provides a new option for users to print a single page of PDF document. In addition, Microsoft also provides "adapt to page", "according to paper size", "original size" and other printing options.
Password manager
Edge 90 provides a free password management tool in the settings, which makes it easy to track password security when online payment, so as to prevent potential vulnerabilities. Users can choose to delete multiple saved passwords in the tool.
Font rendering
In the new browser, Microsoft has improved the font rendering engine to reduce edge blur and improve clarity on some websites.
Auto fill suggestions
The new version of edge can automatically pop up suggestions when users fill in data and forms to help users quickly input saved information. In some cases, if edge can't identify the form information for automatic filling, users can still search for personal information and fill in it quickly in the automatic filling tool.
Browser synchronization
The new version of edge now allows you to search both local and online saved history, previously only local data.
