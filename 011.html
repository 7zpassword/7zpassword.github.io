Huawei disclosed patents for "image recognition method, device, computer readable storage medium and chip"

Recently, Huawei has added a number of patents, including "image recognition method, device, computer readable storage medium and chip" patents.

According to the abstract of the patent, this application provides an image recognition method, device, computer-readable storage medium and chip, which includes: To extract image features to be processed, determine the image to be processed in multiple characters in each character in the image to be processed in multiple frames of each frame image of temporal and spatial characteristics, according to the temporal and spatial characteristics determine its movement characteristic, and identify the image to be processed according to the movement characteristic of multiple characters of group action.

By determining the temporal correlation between the actions of each character and the actions of other characters in the image to be processed, the method can better identify the group actions of multiple characters in the image to be processed.
