Yingchi is developing Hof series DDR5 memory modules

A new ddr5dram chip tray has been released on the Facebook account of galaxy oclab, and the latest hofddr5 memory is expected to be released with Intel's 12th generation core desktop processor by the end of this year or early next year.
Intel's 12 generation core alderlake will be designed with large and small cores. The large core is golden cove and the small core is gracemont. The single thread performance of goldencove large core has been improved by 20%. With the addition of gracemont small core and hardware guided scheduling, the performance under multi thread workload will be twice the original.
Intel's 12th generation core desktop processor supports up to dual channel ddr5-4800 memory, but only the high-end z690 motherboard will provide ddr5 support. According to the news, the sharp dragon 6000 series desktop processor to be released by AMD will not provide ddr5 support. That is to say, the full amount of ddr5 memory will have to wait until the next generation platforms of Intel and AMD.
