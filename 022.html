Microsoft win10 WSL supports running Linux GUI software

The announcement will bring a major update to the win10 WSL Linux subsystem, which will support GUI Graphical interface software running Linux version, officially known as wslg.
Microsoft said that wslg supports running various mainstream Linux GUI software, such as GEDIT text editor and testcafe studio test software.
At the same time, wslg also supports audio and microphone input and output, so some Linux versions of audio software can also be used.
It is worth mentioning that wslg supports GPU acceleration, and the official demonstration runs at 60fps in gazebo 3D software.
Wslg has been supported in the preview version of win10 21364 or above. If WSL is already installed, just enter WSL - update to update to wslg.
